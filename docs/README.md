# What is Kentan?

# Features

- Organizes your test data
- Makes your test data reusable
- Configures test data with ease
- Works with every testing framework ([Jest](https://jestjs.io/), [Mocha](https://mochajs.org/), [Jasmine](https://jasmine.github.io/), etc).
